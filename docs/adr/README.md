# Architecture Decision Records

This repository uses Architecture Decision Records (ADRs) to document important design choices and their rationale.

Each ADR follows the format:
- Context
- Decision
- Consequences

The ADRs in this directory cover the contract platform, TTL design, privacy, governance, caching, and runtime targets.
